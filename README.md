## Requirements
You can install the Requirements by running the command:

```
pip install -r requirements.txt
```

## Installation
```sh
$ git clone https://github.com/alii76tt/iris-identification
$ cd iris-identification
$ pip install -r requirements.txt
$ python manage.py runserver
```
